## Why this journey exists

Single clinics are one-off availability. This journey documents that shorter branch separately so
the map shows only the screens users need for a standalone clinic.

## Key decisions

- The type-of-clinic step is still the public entry point, but this concept hydrates it as a
  single clinic.
- Single clinic creation skips weekdays and closures.
- Times, capacity and appointment length feed the appointments-per-clinic calculation.
