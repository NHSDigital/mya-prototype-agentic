## User story

**As a** site administrator
**I want to** create a standalone single clinic
**So that** appointments can be booked against one-off availability.

## Acceptance criteria

- **Given** I choose a single clinic
  **When** I continue
  **Then** I enter one clinic date.

- **Given** I enter start time, end time, appointment length and capacity
  **When** I continue
  **Then** I see the resulting number of appointments for the clinic.

- **Given** I have completed the steps
  **When** I reach check answers
  **Then** I can review every choice before creating the clinic.
